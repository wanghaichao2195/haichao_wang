leetcode 51 - N-Queens

Haichao's Blog
Nov,06 2019
leetcode DFS backtracking

The original solution using a 2 dimensional arr as temp for backtracking. So the space will be O(n^2)
I improve the space to O(n) using a single arr. chess[i] = j means at ith row, the chess is on jth col.
